# bitlist
rust cli app that shows the set bits of any binary, decimal, or hexadecimal number.  
Much like the bit toggling view of windows 10 calculator

## Usage:
```
bitlist <input type> <number>
```
## Examples:
```
$ bitlist -b 111110000
$ bitlist -h 1234abcde
$ bitlist -d 12344432
```
